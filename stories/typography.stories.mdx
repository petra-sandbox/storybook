# Typography
**Font:** Barlow

**Weights:** 300 (light), 400 (regular), 600 (semibold), 700 (bold), 800 (extrabold), 900 (black)

## **Heading sizes**

Semantic use of headings:
- only use one title (h1) per page, at the top of the page
- Do not skip heading levels e.g.: from h2 to h5 (you can skip headings in the other direction e.g.: from h5 to h2)

This needs to be updated since Tailwind.

| ---- | ---- | ----------- | ----------- |

| Text color | `--text` | #595959 | #EBEBEB |
| Muted or inactive text color | `--muted` | #A2A2A2 | #A2A2A2 |
| Link | `--brand-primary` | #34AEA1 | #34AEA1 |
